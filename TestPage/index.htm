H1--How to Put “Open Command Window Here” Back on the Windows Right-Click Menu

H3-- Open the Registry Editor by hitting Start and typing “regedit.” Press Enter to open Registry Editor, and then give it permission to make changes to your PC.

1. Go to > HKEY_CLASSES_ROOT\Directory\shell\cmd
2. After gaining control of the cmd key, you’re going to make one small change to one of the values inside it. Right-click the HideBasedOnVelocityId value and select the “Rename” command. Just put an underscore ( _ ) at the beginning of the value’s existing name. All we’re doing here is making the value not register.
